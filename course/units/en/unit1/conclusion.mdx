# Conclusion

Congratulations on completing Unit 1! Let's recap what you've learned.

## Key Takeaways

**Skills bridge the gap between general-purpose agents and domain expertise.** Without skills, agents have to figure out specialized workflows from scratch every time. With skills, they can draw on pre-packaged knowledge that encodes best practices, decision logic, and tested templates.

**The SKILL.md format is simple but powerful.** A markdown file with YAML frontmatter is all you need to define a skill. The format is readable by both humans and agents, works across different coding platforms, and plays well with version control.

**Good skills use progressive disclosure.** The main `SKILL.md` file acts as a router — it gives the agent a high-level workflow and points to scripts, references, and templates for deeper detail. This keeps context usage efficient while allowing unlimited depth.

**The ecosystem is growing.** The Hugging Face Skills repository already includes skills for model training, evaluation, dataset creation, CUDA kernel development, and more. And the `upskill` tool lets you generate and evaluate new skills systematically.
