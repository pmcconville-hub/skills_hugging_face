# What Are Agent Skills?

In this unit, you'll learn what agent skills are, why they matter, and how they work. By the end, you'll have written your own skill from scratch and installed skills to use with your coding agent.

## Learning Objectives

After completing this unit, you will be able to:

- Explain why skills are needed and how they improve coding agents
- Describe the `SKILL.md` format and the Agent Skills Specification
- Identify the components of a skill: frontmatter, instructions, scripts, and references
- Explain how progressive disclosure works when agents load skills
- Write a skill from scratch
- Find, install, and use skills from the Hugging Face Hub
- Share a skill with the community

## Unit Overview

This unit is structured as follows:

1. **Why Skills Matter** — The problem skills solve and how they fit into the agent ecosystem
2. **The SKILL.md Format** — The specification that defines how skills are written
3. **Anatomy of a Skill** — A deep dive into frontmatter, instructions, scripts, and references
4. **Progressive Disclosure** — How agents discover, load, and activate skills
5. **Hands-on: Write Your First Skill** — Build a skill from scratch
6. **Hands-on: Find and Install Skills** — Work with the Hub skill ecosystem
7. **Sharing Skills** — How to publish and distribute your skills

Let's begin by understanding why skills matter.
