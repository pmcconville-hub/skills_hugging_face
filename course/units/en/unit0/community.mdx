# Joining the Community

Learning is better together. Here's how to connect with other learners and get help throughout the course.

## Discord

Join the Hugging Face Discord server and head to the skills channel for real-time discussion:

- [Hugging Face Discord](https://discord.gg/huggingface)
- Look for the **#agent-skills** channel

This is the best place to ask questions, share what you're building, and get help from other learners and the HF team.

## Hugging Face Hub

The [hf-skills](https://huggingface.co/hf-skills) organization on the Hub is where community-contributed skills, models, and datasets live. Join the organization to:

- Browse and install skills created by others
- Share your own skills with the community
- Contribute to collaborative projects

Join here: [hf-skills organization](https://huggingface.co/organizations/hf-skills/share/KrqrmBxkETjvevFbfkXeezcyMbgMjjMaOp)

## GitHub

The course materials and skills are open source:

- [huggingface/skills](https://github.com/huggingface/skills) — Official HF Skills repository
- [huggingface/upskill](https://github.com/huggingface/upskill) — Skill generation and evaluation tool

Open issues or pull requests if you find bugs, want to suggest improvements, or have questions about the code.

## Getting Help

If you're stuck on a hands-on exercise:

1. Check the **Troubleshooting** sections in each unit
2. Search the Discord channel — someone may have had the same issue
3. Open a GitHub issue with details about what you tried
4. Ask in the Discord channel and tag `@hf-skills` for visibility

## Sharing Your Work

We encourage you to share your progress as you go through the course. Post on social media with **#AgentSkillsCourse** or share in the Discord channel. It's a great way to get feedback and inspire others.
