# Welcome to the Skills Course

Welcome to the Hugging Face **Skills Course**! This course teaches you how to build, evaluate, and share portable skills that make coding agents expert at specialized AI tasks.

## What You'll Learn

Coding agents like Claude Code, Codex, and Gemini CLI are powerful general-purpose tools, but they often lack deep expertise in specialized domains. Agent skills solve this problem by packaging domain knowledge into portable, reusable instruction sets that any coding agent can follow.

In this course, you'll learn how to:

- Understand the **SKILL.md** format and the Agent Skills Specification
- Write your own skills from scratch
- Find, install, and use skills from the Hugging Face Hub
- Use skills to automate real ML workflows: evaluation, dataset creation, and fine-tuning
- Share your skills with the community

## Who Is This Course For?

This course is for anyone who works with coding agents and wants to make them more effective. Whether you're an ML engineer automating training pipelines, a researcher evaluating models, or a developer building tools for the community, skills give you a way to encode your expertise and share it with others.

## Prerequisites

Before starting this course, you should have:

- Familiarity with **Python** or **JavaScript** and the **command line**
- A [Hugging Face account](https://hf.co/join)
- Access to a coding agent (Claude Code, Codex, Gemini CLI, Cursor, or similar)

No prior experience with agent skills is required.
